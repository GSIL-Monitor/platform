# 宠物信息服务平台

### 目录说明
```
├──.bin.......................开发使用的辅助软件
├──cdn........................开发时模拟的CDN静态资源节点
├──dependencies...............依赖项目
│  ├──entity-interface...........公共依赖接口或公共JavaBean
│  ├──fastjson-master............基于官方源码修改一个bug(未来版本中官方可能会解决)
│  ├──sms........................短信API
│  └──utils......................自定义工具包
├──doc........................说明文档及统一配置文件
├──html.......................前端页面
├──ossfile....................文件空间【模块】
└──user.......................用户【模块】
其他模块待添加
```
